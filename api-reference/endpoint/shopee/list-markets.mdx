---
title: "List Markets"
openapi: "GET /v1/shopee/markets"
description: "List all supported Shopee marketplaces."
---

## Response

<ResponseField name="markets" type="array">
  Array of supported Shopee marketplaces.

  <Expandable title="Market object">
    <ResponseField name="code" type="string">Market code to use in the `market` query parameter (e.g. `"sg"`, `"my"`, `"th"`).</ResponseField>
    <ResponseField name="domain" type="string">Shopee storefront domain (e.g. `"shopee.sg"`).</ResponseField>
    <ResponseField name="name" type="string">Marketplace country name.</ResponseField>
    <ResponseField name="currency" type="string">ISO 4217 currency code used in this market.</ResponseField>
    <ResponseField name="locale" type="string">Default locale for this market.</ResponseField>
    <ResponseField name="country_code" type="string">ISO country code used for proxy geo-targeting.</ResponseField>
  </Expandable>
</ResponseField>
