---
title: "Get Listing Detail"
openapi: "GET /v1/immobiliare/listings/{id}"
description: "Full Immobiliare listing detail — price + €/m², surface, energy class, condo fees, GPS, all photos, per-unit breakdown, agency and agent contacts."
---

Fetch a single listing's complete detail by its numeric id (from
[`/search`](/api-reference/endpoint/immobiliare/search) or a portal URL such as
`immobiliare.it/annunci/121584766/`). The response is the full `Listing`
object — everything the search card has, plus the full description, all photos
and the complete per-unit `properties[]` breakdown.

## Path Parameters

<ParamField path="id" type="integer" required>
  Immobiliare listing id, e.g. `121584766`.
</ParamField>

## Query Parameters

<ParamField query="market" type="string" default="it">
  Market the listing belongs to. One of `it`, `es`, `gr`, `lu`.
</ParamField>

<CodeGroup>

const res = await fetch(

);
const listing = await res.json();
```

```
