---
title: "Get Video Detail"
openapi: "GET /v1/tiktok/videos/{video_id}"
description: "Full metadata for a single TikTok video or photo slideshow."
---

<Tip>
  Pass the author's `username` to skip the internal oEmbed author lookup for a faster, cheaper fetch.
</Tip>

<Note>
  Each request costs **5 credits**. Failed requests are not charged.
</Note>
