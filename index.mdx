## What is ScrapeBadger?

ScrapeBadger is a unified API for accessing Twitter/X data. Instead of managing scraping infrastructure, proxies, and session rotation yourself, make simple REST API calls and get structured JSON data back.

**Base URL:** `https://scrapebadger.com/v1`

```bash
curl -X GET "https://scrapebadger.com/v1/twitter/users/elonmusk/by_username" \
  -H "x-api-key: YOUR_API_KEY"
```
